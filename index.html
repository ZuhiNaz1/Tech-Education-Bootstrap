Flexbox: 1. It is one dimensional layout. 2. It focuses on content flow(content first). 3. It is small design implementation. Grid: 1. It is multidimensional layout. 2. It focuses on content placement(layout first). 3. It is complex design implementation.

Bootstrap: 1. It comes with a set of pre-styled responsive,mobile-first-components that posses a definite UI kit. 2. It is efficient and saves a lot of time. Tailwind: 1.It offers pre-designed widgets to build a site from scratch with fast UI development. 2. It is relatively new and there is still room to grow in terms of its community.

CSS box model: In CSS, the term 'box model' is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML elements.

Semantic tag: A semantic element clearly describes its meaning to both the browser and the developer. The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. Example : form tag, table tag, article tag etc.
